# Setting Up Claude Properly

A comprehensive setup guide for Claude Code, Cowork, and Chat — written for non-technical business decision makers. Covers configuration, projects, skills, MCP connectors, team-specific tool recommendations, and best practices.

**Live site:** https://setting-up-claude-properly.vercel.app

## What's in here

- **`index.html`** — the main guide site (dark terminal aesthetic, inspired by skillsmp.com)
- **`Claude-Code-Setup-Guide.md`** — markdown version of the same guide
- **`claude-code-setup-guide.html`** — shareable standalone HTML version
- **`installed-skills-reference.md`** — reference list of installed skills
- **`pre-install-checklist.md`** — checklist to follow before installing any new skill, repo, MCP, or tool
- **`mcp-install-instructions.md`** — how to install MCP connectors via Claude Desktop
- **`decisions.md`** — log of design and structural decisions made on this guide
- **`CLAUDE.md`** — project rules for Claude (auto-loaded)
- **`Context.md`** — living context, updated each session
- **`.claude/`** — Claude config (hidden)

## Who this is for

Anyone setting up Claude (Desktop, Code, or Cowork) without a developer background. Particularly useful for business leaders, agency staff, and small business owners who want practical setup guidance in plain English.

## Project mode breakdown

The guide is structured as three mini-sites for the three Claude modes:

- **Chat** — Claude.ai conversational mode
- **Cowork** — collaborative project workspace with connectors
- **Code** — full-power coding mode with skills, MCPs, hooks

Plus a shared **All Modes** section covering universal setup (CLAUDE.md, Context.md, memory, security, etc.).

## Deployment

Deployed to Vercel from the `main` branch of this repository. Any push to `main` auto-deploys.
